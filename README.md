### 掌上教务 React项目



# 架构说明

## 1 文件结构说明

- src 为代码目录
    - _tests_ 单元测试目录
    - components 公用组件目录
    - img 静态资源图片
    - less 公共样式文件
    - pages 页面和业务组件
        - community 社区相关业务包
    - utils 公共库
    - public 为静态资源文件夹
